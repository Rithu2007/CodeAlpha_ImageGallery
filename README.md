# GalleryX - Personal Photo Gallery

GalleryX is a modern, client-side web application that enables users to securely store, organize, and view their personal photos through a premium and responsive user interface.

## GitHub Repository

🔗 https://github.com/Rithu2007/CodeAlpha_ImageGallery

🌐 Live Demo: https://rithu2007.github.io/CodeAlpha_ImageGallery/

## Overview

GalleryX provides a seamless experience for managing personal photo collections. Users can sign up, sign in, upload images, organize memories, view analytics, and interact with media through an elegant glassmorphic interface.

The application is completely frontend-based and uses browser localStorage for data persistence, eliminating the need for external databases or backend services.

## Data Storage

GalleryX uses browser localStorage to store user information and uploaded images. Files are converted into Base64 Data URLs and managed entirely on the client side, ensuring fast performance and a lightweight architecture.
